PayPal QA Testing Project

Software Quality Assurance Documentation

Project Overview

This QA testing project focuses on the PayPal payment system, ensuring that key features such as sending and receiving payments are tested thoroughly. The goal is to validate functionality, usability, and security through a series of structured testing phases.

Tools Used

  • Jira – Bug Tracking
  • Selenium – Automated Testing
  • Postman – API Testing
  • TestLink – Test Case Management
  • Microsoft Excel/Word – Documentation
